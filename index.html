RL Insider
Frontend Development
Implemented the frontend for RL Insider, the most accurate price tracker for Rocket League Items. Tested on various platforms, browsers and devices to ensure stability and performance were up to industry standards. Rocket League Insider became the leading community price reference page with over two million unique users by May 2018.
Popps.js
Javascript Canvas Library
Popps.js is a Javascript Library I wrote in order to simplify the HTML5 Canvas. It powers my online games and visual projects and must be stable and lightweight. I learned how to write generalized, performance focused code from this project.
Down the Dungeon
Javascript Dungeon Crawler
Down the Dungeon is a browser based dungeon crawling game. It focuses on performance, asthetics, and fun gameplay. Down the Dungeon runs on Popps.js, with a custom engine, and attempts to push the limits of browser gaming. From this project I learned countless algorithms, design patterns, and most importantly, how to code smarter, not harder.
Color Picker
For Designers and Developers
This custom color picker provides a quick and visual way to select colors and copy/paste them to your code. It includes a unique hue wheel, a large panel to view the selected color, and the ability to copy the 3 most common color representations to your clipboard. This project taught me how color is represented on computers, as well as their relationships with each other.
