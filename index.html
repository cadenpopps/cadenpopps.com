Professional and personal projects I've worked on. Please view the code on GitHub and click the links to explore the website for each project.
RL Insider
Freelance Frontend Development
RL Insider is the most accurate price tracker for Rocket League Items. This website was my first paid freelance work, and it attracted over 1 million users in a matter of months. This project taught me to work in a team with another developer and a designer.
Popps.js
Javascript Canvas Library
Popps.js is a Javascript Library I wrote in order to simplify the HTML5 Canvas. It has been used and tested in all of my online games, and all of the code is written from scratch. I learned how to write generalized, performance focused code from this project.
Down the Dungeon
Javascript Dungeon Crawler
Down the Dungeon is a one of a kind browser game. It focuses on performance, asthetics, and fun gameplay. I wrote Down the Dungeon in Javascript, including a custom game and display engine, in order to push the limits of browser gaming. While it is not yet compelete, this project has taught me so much: countless algorithms, design patterns, and most importantly, how to code smarter, not harder.
Color Picker
For Designers and Developers
This color picker provides a quick and visual way to select colors and transport them to your code. It includes a unique hue wheel, a large panel to view the color you selected, and the ability to copy 3 common color representations to your clipboard. This project taught me how color is represented on computers, as well as their relationship with one another.
